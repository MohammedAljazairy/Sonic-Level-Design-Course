# Sonic the Hedgehog — 3D Unity Remake

Ever played the classic 2D Sonic games on the Sega Genesis? This project brings that experience into 3D! I recreated the iconic **Green Hill Zone, Act 1** from the original Sonic the Hedgehog as a fully playable 3D Unity game.

## 🎮 Gameplay
Play as Sonic and race through a 3D reimagining of Green Hill Zone, staying true to the feel of the original.

## 🕹️ Controls
| Action | Key |
|---|---|
| Move | WASD / Arrow Keys |
| Jump | Space |
| Crouch | F |
| Spin Dash | F + Space |
| Camera | Mouse |

## ▶️ How to Run
This project currently runs in the **Unity Editor only**.

1. Clone the repository
2. Open the project in Unity 6000.3.12
3. Open the `GreenHillZone` scene and press **Play**
